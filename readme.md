# 🏥 Rural Health Triage Assistant — MVP

Offline AI symptom checker for rural health workers and villagers.
Runs on any machine, no internet needed after setup.

## Quick Start (3 commands)

```bash
pip install -r requirements.txt

python train_model.py      # trains model (~10 sec), creates models/ folder

python app.py              # opens UI at http://localhost:7860
